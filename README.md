# IPL Data Analysis using NumPy & Pandas

##  Project Overview
This project focuses on **Indian Premier League (IPL) data analysis** using **Python (NumPy & Pandas)** in **Anaconda / Jupyter Notebook**.  
The goal is to perform **data cleaning, analysis, aggregation, and visualization** to extract meaningful insights from IPL match data.

---

##  Tools & Technologies Used
- Python
- Anaconda
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib / Seaborn

## Analysis Performed
- Data loading and inspection  
- Handling missing values  
- Data filtering and sorting  
- GroupBy operations  
- Aggregation using Pandas  
- Basic numerical operations using NumPy  
- Visualization of key insights

## How to Run This Project
1. Install Anaconda  
2. Open Anaconda Navigator → Launch Jupyter Notebook  
3. Open `IPL Project data analysis.ipynb`  
4. Run all cells to view the analysis
